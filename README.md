> `JavaScript` `k6` `Prometheus` `Grafana` `Docker Compose` `GitHub Actions`

Fully containerized performance testing suite running **5 test types** (smoke → load → stress → spike → soak) against an OpenAPI mock server. Metrics flow from k6 into **Prometheus** via Remote Write and are visualized in real-time **Grafana dashboards**. Zero external dependencies — spins up with a single `docker compose up`.
